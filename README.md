# Automated Image Processing using DevOps CI Pipeline

## Description
This project demonstrates automated image processing using Python and OpenCV integrated with a GitHub Actions CI pipeline.

## Features
- Color Pop Effect
- Neon Glow Edge Effect
- Automatic image detection
- CI pipeline on every push

## Tools
- Python 3
- OpenCV
- GitHub Actions
- PyTest

## How to Run
1. Place images in the `input/` folder
2. Run:
   ```bash
   python src/main.py
